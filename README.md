# Project-CFE: Quản lí thu/chi

# Mục tiêu dự án: 
Tạo ra website quản lí thu/chi cá nhân với những tính năng như: xem lịch sử, thêm/sửa/xóa khoản thu/chi, dashboard tổng quan...


# Cài đặt dự án
- Bước 1: Tải Node.js trên mạng
- Bước 2: Clone dự án ra vscode
- Bước 3: chạy lần lượt các lệnh sau trong Terminal
    npm init -y
    npm install lite-server --save-dev
    npm start
